# How To Use SendEvents

## Install PreReqs

As root

<pre>
yum -y install epel-release
yum -y install git
yum -y install maven
</pre>

## Clone this repo  
As orginary user

<pre>
git clone https://github.com/david618/createRouteData
</pre>

## Build 

<pre>
cd createRouteData
mvn install
</pre>

Should end with BUILD SUCCESS
